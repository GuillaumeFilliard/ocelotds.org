OcelotController

ocelotController is the controller of Ocelot…
An instance is created on loading of page you can access it by “ocelotController”.
You can do some actions with it.

Method addOpenListener


ocelotController.addOpenListener(openHandler:Function())

Add handler when ws is open, this is retroactive

Method addCloseListener


ocelotController.addCloseListener(closeHandler:Function())

Add handler when ws is close

Method addErrorListener


ocelotController.addErrorListener(errorHandler:Function())

Add handler when ws throws an error

Method close


ocelotController.close()

Close the Websocket

Method open


ocelotController.open()

After close, reup the connection, redo existants subscriptions

Attribute status


ocelotController.status [Readonly]

Return the state of websocket : ‘CONNECTING’, ‘OPEN’, ‘CLOSING’, ‘CLOSED’

Method cacheManager.addEventListener


ocelotController.cacheMAnager.addEventListener(type:String, listener:Function(evt:Event))

Add handler for add or remove entry in cache

Method cacheManager.clearAll


ocelotController.cacheMAnager.clearAll()

Remove all entries in cache
