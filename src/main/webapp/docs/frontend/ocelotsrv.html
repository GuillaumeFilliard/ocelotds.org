OcelotServices

OcelotService is service serve by Ocelot allow doing some actions

You have to instance it for use it.

Create


var srv = new OcelotServices();

OcelotServices is a standard services like every service like you can create.

Method getLocale


srv.getLocale():String

Get user locale from server Exemple


srv.getLocale().then(function(loc) {
   doSmethingWithLocale(loc);
});

Method setLocale


srv.setLocale(loc:String)

set user locale to server Exemple


srv.setLocale({"language":"fr", "country":"FR"});
