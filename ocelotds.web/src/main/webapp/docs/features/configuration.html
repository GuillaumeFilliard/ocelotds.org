Configuration :

Configuration is completly optional.

Configure stacktrace length

You can set the length of stacktrace. (default value 50)

In web.xml

>context-param<
   >param-name<ocelot.stacktrace.length>/param-name<
   >param-value<10>/param-value<
>/context-param<

Use CDI producer

Create simply a cdi producer qualifed by annotation
@org.ocelotds.configuration.annotations.OcelotConfiguration(OcelotConfigurationName.STACKTRACELENGTH).

@Produces @OcelotConfiguration(OcelotConfigurationName.STACKTRACELENGTH) String getStacktraceConfiguration() { return "20"; }
