Configuration :

Configure Secure websocket

You can use websocket secure.

Set WSS in web.xml

>context-param<
   >param-name<ocelot.websocket.secure>/param-name<
   >param-value<true>/param-value<
>/context-param<

Use CDI producer

Create simply a cdi producer qualifed by annotation
@org.ocelotds.configuration.annotations.OcelotConfiguration(OcelotConfigurationName.SECURE).

@Produces @OcelotConfiguration(OcelotConfigurationName.SECURE) String getSecureConfiguration() { return "true"; }

Configure stacktrace length

You can set the length of stacktrace.

In web.xml

>context-param<
   >param-name<ocelot.stacktrace.length>/param-name<
   >param-value<10>/param-value<
>/context-param<

Use CDI producer

Create simply a cdi producer qualifed by annotation
@org.ocelotds.configuration.annotations.OcelotConfiguration(OcelotConfigurationName.STACKTRACELENGTH).

@Produces @OcelotConfiguration(OcelotConfigurationName.STACKTRACELENGTH) String getStacktraceConfiguration() { return "20"; }
